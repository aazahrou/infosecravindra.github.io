Sublime Text Cheatsheet

Sublime Textis awesome, I have been using it for everything and started taking notes about its features so here is a cheatsheet I have created for sublime

  • Toggle sidebar:CTRL + K, CTRL + B
  • Open command palette:CTRL + SHIFT + P
  • Open fuzzy file search:CTRL + P
  • Fold/Hide attribute in HTML/XML:CTRL + K, CTRL + T
  • Unfold/Unhide everything:CTRL + K, CTRL + J
  • Convert selection to UPPERCASE:CTRL + K, CTRL + U
  • Convert selection to lowercase:CTRL + K, CTRL + L
  • Wrap selection withing tag:ALT + SHIFT + W
  • Distraction free mode:SHIFT + F11
  • Redo last action :CTRL + Y
  • Paste from history:CTRL + K, CTRL + V
  • Paste and indent:CTRL + SHIFT + V
  • Indent:CTRL + ]
  • Unindent:CTRL + [

Selection

  • Select line:CTRL + L
  • Select selected word’s next entry:CTRL + D
  • Select both ways from cursor:CTRL + SHIFT + SPACE
  • Select tag text then tag itself then so on:CTRL + SHIFT + A
  • Select everything within same indent level:CTRL + SHIFT + J
  • Close current tag:ALT + .

Navigation

  • Goto line:CTRL + G

Folding

  • Fold tag:CTRL + SHIFT + [
